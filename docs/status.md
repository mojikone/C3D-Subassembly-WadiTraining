## Sampling Update

Replaced the fixed 10-point sampling prototype:
- packet now includes `AP1` through `AP250` as possible auxiliary sample points
- each sample is guarded by a SAC `FlowDecision`
- sample spacing remains 1 m using `DeltaXOnSurface`
- decision logic uses `ThalwegOffset` when valid, otherwise `MaxScanDistance`
- scan sample autolinks are disabled for now; the sample points are auxiliary only
- `CreateLoop` was not used because SAC loop geometry copies a link vector; it does not re-run `DeltaXOnSurface` on irregular ground
