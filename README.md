# C3D Subassembly — Wadi Training Levee

Civil 3D 2026 .NET (C#) subassembly, packaged as an importable `.pkt`, that draws a one-side
terrain-adaptive wadi levee and detects terrain trend breaks (concave = protect, convex = debug).
